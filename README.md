## Задача 1

### Текст задачи

Сформировать список из чисел, на 1 больших, чем вводимые значения.

### Алгоритм решения

1)Запросить у пользователя ввод чисел через пробел
2)Прочитать строку с консоли
3)Разбить строку на массив строк по пробелам

5)Применить функцию List.map (fun x -> x + 1) для создания нового списка
6)Вывести исходный список и список с увеличенными числами

# Задание 3

## Задача 1

### Текст задачи
Создайте собственные функции для выполнения основных операций над списками (добавление/
удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

I. Ввод начального списка
	1)На экран выводится приглашение "Введите числа через пробел: "
	2)программа ожидает, пока пользователь введет строку и нажмет Enter
	3)Введенная строка сохраняется
	4)Строка разбивается на отдельные части по пробелам
	5)Каждая непустая часть преобразуется в число
	6)Из полученных чисел формируется список initialList
	7)На экран выводится начальный список
II.Запуск меню
	1)Функция task3 вызывает функцию menu и передает ей список initialList
	2)Управление переходит в функцию menu
III.Работа меню
	1)Выводится меню с вариантами действий
	2)Выводится приглашение "Выберите действие: "
	3)Программа ожидает ввод пользователя
IV.Обработка выбора
	1)При 1
	-Выводится приглашение "Введите число: "
	-Пользователь вводит число
	-Вызывается функция addNumber с текущим списком и числом
	-На экран выводится сообщение о добавлении числа
	-Выводится обновленный список
	-Функция menu вызывает саму себя (рекурсия) и передает новый список
	-Программа возвращается к ШАГУ 5 с новым списком
		
2)При 2
	-Выводится приглашение "Введите число для удаления: "
	-Пользователь вводит число
	-Запоминается длина текущего списка
	-Вызывается функция removeNumber с текущим списком и числом
	-Функция removeNumber создает новый список без удаляемого числа
	-Сравнивается длина нового списка со старой
	-Выводится сообщение "Число * удалено" или "Число * не найдено"
	-Выводится обновленный список
	-Функция menu вызывает саму себя (рекурсия) и передает новый список
	-Программа возвращается к ШАГУ 5 с новым списком
	3)При 3
	-Выводится приглашение "Введите число для поиска: "
	-Пользователь вводит число
	-Вызывается функция findNumber с текущим списком и числом поиска
	-Функция findNumber рекурсивно проверяет элементы списка
	-Возвращается результат true или false
	-Выводится сообщение "Число * есть в списке" или "Числа * нет в списке"
	-Функция menu вызывает саму себя и передает тот же список
	-Программа возвращается к ШАГУ 5

4)При 4
	-Выводится приглашение "Введите индекс: "
	-Пользователь вводит индекс 
	-Вызывается функция getNumberByIndex с текущим списком и индексом *
	-Функция getNumberByIndex рекурсивно проходит по списку
	-Возвращается значение элемента или -1 
	-Если результат -1, выводится "Нет элемента с индексом *"
	-Иначе выводится "Элемент[*] = *"
	-Функция menu вызывает саму себя и передает тот же список
	-Программа возвращается к ШАГУ 5
	5)При 5
	-Выводится приглашение "Введите второй список через пробел: "
	-вводит второй список 
	-Вызывается функция readList для обработки введенной строки
	-Получается второй список 
	-Вызывается функция joinLists с текущим и вторым списком
	-Создается новый объединенный список
	-Выводится "Списки соединены"
	-Выводится "Результат"
	-Функция menu вызывает саму себя и передает новый список
	-Программа возвращается к ШАГУ 5
